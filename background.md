## Project Aegis

### 組織概要
- **設立**: 2018年
- **本社**: サンフランシスコ ダウンタウン（47階建てビル）
- **従業員数**: 約500名（セキュリティ部門は50名）
- **表向きの事業**: サイバーセキュリティコンサルティング
- **真の目的**: 世界のサイバー空間における「均衡」の維持
- **守護対象**: 電力網、水道システム、金融ネットワーク、交通管制システム

### 設立の経緯

2017年、世界は「ダークネット・クライシス」と呼ばれる大規模なサイバー攻撃を経験した。複数の国家インフラが同時に攻撃され、世界経済は一時的に麻痺状態に陥った。この事件の背後には、統制の取れていない複数のハッカー集団の存在があった。

事件後、主要国の情報機関の極秘会議で、ある革新的な提案がなされた。「管理された対立」による秩序の維持。つまり、防御側と攻撃側の両方を密かにコントロールすることで、予測不可能な第三者による破滅的な攻撃を防ぐというものだった。

### 創設者たち

**サラ・ミッチェル（CEO）**
- 元NSAサイバー司令部副司令官
- 「ダークネット・クライシス」の際、被害を最小限に抑えた立役者
- 冷静沈着で、感情を表に出さない「アイス・クイーン」の異名を持つ

**マイケル・リー（CTO）**

- 技術至上主義者で、「完璧なシステム」の構築に人生を捧げている

### 組織文化

Project Aegisの社内には、独特の文化が根付いている。

4. しかし、我々の使命は永遠である

この信条は、全従業員が入社時に暗唱させられる。しかし、その裏にある「均衡」の真実を知る者は、上層部のごく一部だけだった。

### Protocol Sevenの真実

Protocol Sevenは、表向きは「緊急時の最終防衛システム」として開発された。大規模なサイバー攻撃に対し、攻撃者のシステムを逆に制圧するためのものとされていた。

しかし実際は、これは「均衡」システムの最終保険だった。もしShadow Collectiveが制御不能になった場合、あるいは第三の勢力が台頭した場合、世界のインフラを一時的に「凍結」し、リセットするためのものだった。

設計に関わったのは、マイケル・リーを含む5人のエンジニア。しかし、バックドアを仕込んだのは、その中の誰か一人。その人物は、今も組織内に潜んでいる。

### 隠された真実
- Shadow Collectiveとの「協定」により世界の均衡を維持
- 防御と攻撃の相互作用により、世界のセキュリティレベルを向上
- 内部に意図的にバックドアを仕込んだ人物が存在
- AEGIS_COMMANDというコードネームで極秘通信を実施

## Shadow Collective

### 組織概要
- **活動開始**: 2020年頃（公式には）
- **構成**: 国際的なハッカー集団
- **表向きの目的**: 
  - 企業秘密の窃取と売買
  - 重要インフラへの攻撃
  - 暗号通貨による資金洗浄
- **真の役割**: Project Aegisと協調し、サイバー空間の均衡を維持
- **特徴**: 高度に組織化され、各国に細胞組織を持つ

### 真の起源

Shadow Collectiveの起源は、Project Aegisの設立とほぼ同時期に遡る。「均衡」システムの立案者たちは、信頼できる「敵」が必要だった。そこで、各国の情報機関から最も優秀で、かつ最も統制の取れたハッカーたちを集め、秘密裏に組織を作った。

初期メンバーの多くは、自分たちが「管理された敵」であることを知らなかった。彼らは、真に世界の不正と戦っていると信じていた。しかし、リーダー格の人物たちは、やがて真実に気づき始めた。

### 主要メンバー

**"Cipher"（サイファー）**
- リーダー格、正体不明
- 一説によると、元MI6のサイバー戦専門家
- 「均衡」システムの熱烈な信奉者だったが、最近その信念に揺らぎが生じている
- 座右の銘: "In chaos, we find purpose"（混沌の中に、我々は目的を見出す）

**"Zero"（ゼロ）**
- 技術責任者、元MIT教授
- アレックスの大学時代の恩師という噂もある
- Protocol Sevenの潜在的な危険性に最初に気づいた人物
- 過激派と主流派の間で板挟みになっている

**"Phoenix"（フェニックス）**
- オペレーション担当、元ロシア軍サイバー部隊
- 冷徹な戦術家として知られる
- 「均衡」など関係なく、純粋に破壊を楽しむタイプ
- 過激派のシンパだが、まだ旗幟を鮮明にしていない

**"Ghost"（ゴースト）**
- 情報収集・諜報担当
- 性別、国籍、年齢、すべて不明
- デジタルの亡霊と呼ばれ、痕跡を一切残さない
- アレックスの動きを最初に察知した人物

### 内部分裂

2024年後半、Shadow Collective内部で大きな変化が起きた。若い世代のハッカーたちが、組織の「生ぬるい」活動に不満を持ち始めたのだ。

**主流派**: 「均衡」システムを支持し、Project Aegisと協調
- Cipherを中心とする古参メンバー
- 世界の安定を重視
- 「管理された混沌」を信奉

**過激派**: 均衡を拒否し、世界のインフラ完全支配を目論む
- 若手メンバーが中心
- 真の自由と無秩序を求める
- Protocol Sevenを「解放の鍵」と見なす
- リーダーは"Anarchy"（アナーキー）というコードネームの人物

### 活動実績
- 2021年: 大手銀行への攻撃（10億ドルの被害）※均衡維持の一環
- 2022年: 政府機関のデータベース侵入 ※計画的な演習
- 2023年: 暗号通貨取引所への大規模攻撃
- 2024年12月15日: Phase 3の開始
- 2025年6月: 過激派がProtocol Sevenの奪取を試みる

### 組織の哲学

Shadow Collectiveには、独自の哲学体系がある。

**「The Shadow Doctrine」（影の教義）**
1. 情報は自由であるべきだ
2. しかし、自由には責任が伴う
3. 我々は混沌の中に秩序を作る
4. 破壊は創造の始まりである

この教義の解釈を巡って、主流派と過激派の対立が深まっている。

## Protocol Seven

### 概要
- Project Aegisのシステム深層に組み込まれたバックドア
- 設計段階から意図的に仕込まれた
- 世界中の重要インフラを一斉に制御可能

### 開発の経緯

2019年、Project Aegisは「Project Titan」というコードネームで、究極の防御システムの開発を開始した。このシステムは、あらゆるサイバー攻撃を無効化し、攻撃者を逆に制圧する能力を持つはずだった。

しかし、開発の過程で、ある人物が密かに「保険」を組み込んだ。それがProtocol Sevenだった。この人物は、「均衡」システムそのものに疑問を抱き、いつか誰かがこのシステムを破壊する必要が来ると考えていた。

### 技術仕様

Protocol Sevenは、7つのレイヤーから構成される：

1. **Network Layer**: 全世界のルーターに潜むスリーパーコード
2. **Protocol Layer**: 通信プロトコルに仕込まれたバックドア
3. **Application Layer**: 主要なアプリケーションの隠し機能
4. **Data Layer**: データベースシステムの管理者権限
5. **Hardware Layer**: ファームウェアレベルの制御機能
6. **Human Layer**: ソーシャルエンジニアリング用の心理プロファイル
7. **Quantum Layer**: 量子暗号を突破する未来技術（理論段階）

### 能力
- 電力網の停止
- 金融システムの麻痺
- 交通管制の大混乱
- 「デジタル・アポカリプス」を引き起こす力

### 現状
- Shadow Collectiveの過激派が奪取を試みている
- アレックスが防御メカニズムを構築
- 解読者の選択により無力化可能

### バックドアを仕込んだ人物

この人物の正体は、物語の最後まで明かされない。しかし、いくつかの手がかりがある：
- Protocol Sevenの設計に関わった5人のうちの1人
- 「均衡」システムに深い疑念を抱いている
- 高度な暗号技術と哲学的思考を併せ持つ
- アレックスに希望を託している

## 三つの選択

### 真実の公表
- 両組織の欺瞞を世界に暴露
- インターネットの分断と新秩序の構築
- 痛みを伴うが必要な変革

この選択をした場合、世界は一時的に大混乱に陥る。しかし、それは新たな透明性の時代の始まりでもある。各国は独自のセキュリティシステムを構築し、真の意味での分散型セキュリティが実現する。

### 現状維持
- Project Aegisの一員として内部改革
- 偽りの平和の継続
- 過激派の脅威は残存

この選択は、最も現実的だが、最も困難な道でもある。内部からシステムを変えるには、長い時間と忍耐が必要だ。そして、常に発見と粛清のリスクと隣り合わせだ。

### 第三の道
- 両組織から独立した新たな守護者

- アレックスが密かに望んでいた選択

これは、アレックスが本当に望んでいた未来。権力や組織に頼らず、個人の良心と技術力だけで世界を守る。それは、かつてMIT時代に夢見た「真の倫理的ハッカー集団」の実現でもある。
